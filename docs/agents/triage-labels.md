# 分类标签

技能使用五个标准的分类角色。本文件将这些角色映射到本仓库 Issue 追踪器中实际使用的标签字符串。

| 技能中的标签          | 本仓库的标签    | 含义                              |
| ---------------------- | --------------- | --------------------------------- |
| `needs-triage`         | `needs-triage`  | 维护者需要评估此 Issue            |
| `needs-info`           | `needs-info`    | 等待报告者补充更多信息            |
| `ready-for-agent`      | `ready-for-agent` | 描述完整，可由 AI Agent 接手     |
| `ready-for-human`      | `ready-for-human` | 需要人类开发者实现                |
| `wontfix`              | `wontfix`       | 不会处理                          |

当技能提到某个角色（例如"应用 AFK-ready 分类标签"）时，使用本表中对应的标签字符串。
